## starScore 星星评分

```javascript
$('selector').starScore({<br/>
  length : 5,<br/>
  size : '20px',<br/>
  sumScore : 5,<br/>
  step : .1,<br/>
  emptyImgUrl : 'images/star-empty.png',<br/>
  fullImgUrl : 'images/star-full.png',<br/>
  prompts : ['很差!','差!','一般!','还可以!','非常好!'],<br/>
  readonly : false<br/>
});<br/>
```

```html
<input data-star="true" data-length="5" data-size="20px" /><br/>
```
